# 스마트 주차장 찾기

순천 지역 주차장의 실시간 주차 가능 대수를 확인하고, T맵/카카오맵/네이버지도로 바로 길안내를 시작할 수 있는 웹 앱입니다.

## 주요 기능

- 🗺️ 지도에 주차장 위치와 주차 가능 대수 표시
- 🎨 색상으로 주차 상태 구분 (여유/보통/만차)
- 🚗 T맵, 카카오맵, 네이버지도로 바로 길안내 연동
- 📱 반응형 디자인 (모바일 지원)
- ⏱️ 실시간 업데이트 시뮬레이션

## 기술 스택

- HTML5
- CSS3
- JavaScript (ES6+)
- Leaflet.js (지도 라이브러리)

## 로컬 실행

1. 파일 다운로드
2. `index.html` 파일을 브라우저로 열기

## 배포

Netlify, Vercel 등 정적 사이트 호스팅 서비스에 배포 가능합니다.
